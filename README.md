# cmake-arm-cortex-m4

An example repository of using CMake with the arm-none-eabi-gcc compiler for an Arm Cortex M4 processor.

# Building

## Debug

cmake --preset debug
cmake --build --preset debug

## Release

cmake --preset release
cmake --build --preset release
